# 📊 Data Analytics Project

## Overview

This project demonstrates a complete end-to-end Data Analytics workflow, starting from data collection and cleaning to data visualization and reporting. The project focuses on extracting meaningful insights from raw data using Python, SQL databases, and Power BI.

## Project Objectives

* Load and process the dataset efficiently.
* Perform Exploratory Data Analysis (EDA).
* Clean and transform the data for accurate analysis.
* Execute SQL queries using multiple database management systems.
* Develop an interactive Power BI dashboard.
* Prepare a detailed analytical report.
* Present findings through a professional PowerPoint presentation.

## Dataset

The dataset is imported into Python for preprocessing and analysis. Various data cleaning and transformation techniques are applied before storing and querying the data in SQL databases.

## Project Workflow

### 1. Data Loading

* Import the dataset into Python.
* Verify data types and structure.
* Inspect rows and columns.

### 2. Exploratory Data Analysis (EDA)

* Analyze data distribution.
* Identify trends and patterns.
* Detect missing values and outliers.
* Create statistical summaries and visualizations.

### 3. Data Cleaning

* Handle missing values.
* Remove duplicate records.
* Correct inconsistent data.
* Convert data types where necessary.
* Prepare the dataset for analysis.

### 4. SQL Analysis

Execute analytical queries using:

* PostgreSQL
* MySQL
* SQL Server

Tasks include:

* Aggregations
* Grouping
* Joins
* Ranking

* Common Table Expressions (CTEs)
